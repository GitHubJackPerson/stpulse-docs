# Тарифы STPulse

Эта инструкция помогает выбрать подходящий тариф STPulse, понять, что входит в каждый план, какие есть дополнительные опции и как оформить подписку или тестовый доступ.

---

## 1. Доступные тарифы

На странице тарифов представлены три основных плана:

### **3 месяца — “Лёгкий старт”**
Подходит для новых пользователей, которые хотят протестировать сервис в реальной работе.

### **6 месяцев — “Продуманный выбор”**
Сбалансированный тариф для стабильного использования сервиса.

### **12 месяцев — “Максимум выгоды”** *(популярный)*
Самая выгодная цена в пересчёте на месяц.

---

## 2. Что включено во все тарифы

По умолчанию каждый тариф включает:

- Полную финансовую аналитику
- Поддержку одного магазина
- Поддержку одного пользователя

!!! tip "Обратите внимание"
    Во все тарифы включена **финансовая аналитика для одного магазина и одного пользователя**.

При необходимости можно добавить больше магазинов и сотрудников (см. раздел «Дополнительные опции» ниже).

---

## 3. Тестовый доступ (7 дней)

Все пользователи могут активировать **тестовый доступ на 7 дней**:

- полный функционал STPulse, без ограничений,
- возможность подключить магазины,
- загрузка данных и отчётов,
- работа всех разделов: Финансы, Аналитика, Поставки, Реклама, Отчёты.

Чтобы начать тестовый период:

1. Перейдите на главную страницу STPulse.  
2. Нажмите кнопку **«Тестовый доступ»**.  
3. Создайте аккаунт и подключите магазин.

По завершении теста вы сможете выбрать платный тариф.

---

## 4. Выбор тарифа

1. Откройте страницу тарифов.  
2. Ознакомьтесь с доступными планами.  
3. Нажмите **«Выбрать»** под подходящим тарифом.  
4. Перейдите к оформлению заказа.

Если вы уже на тестовом периоде, система предложит оформить подписку без потери данных.

---

## 5. Настройка дополнительных опций

Дополнительные функции позволяют расширить возможности STPulse:

### **Автопродление подписки**
- включено по умолчанию;
- после окончания срока подписка продлевается автоматически.

### **Управление поставками** *(+3 000 ₽/мес)*
Контроль остатков и движений товаров в разрезе складов и кластеров.

### **Управление акциями** *(+1 500 ₽/мес)*
Помогает анализировать эффективность акций и быстро принимать решения.

### **Дополнительный магазин** *(+5 000 ₽/мес)*
Базовый тариф включает один магазин.  
Каждый дополнительный магазин оплачивается отдельно.

### **Дополнительный сотрудник** *(+1 000 ₽/мес)*
Базовый тариф включает одного пользователя.  
Дополнительные сотрудники оплачиваются отдельно.

function calc() {
    const planSelect = document.getElementById("plan");
    const planPerMonth = parseInt(planSelect.value, 10) || 0;
    const months = parseInt(planSelect.selectedOptions[0].dataset.months, 10) || 1;

## 6. Промокоды

Если у вас есть промокод:

1. Введите его в поле **Промокод**.  
2. Нажмите **«Применить»**.  
3. Система автоматически пересчитает стоимость заказа.

---

## 7. Оформление подписки

1. Проверьте выбранный тариф и дополнительные опции.  
2. Ознакомьтесь с итоговой суммой.  
3. Поставьте галочку принятия условий оферты.  
4. Выберите способ оплаты:
   - **Перейти к оплате** — онлайн-оплата банковской картой,
   - **Получить счёт для оплаты** — для ИП и юридических лиц.

После оплаты тариф активируется автоматически.

---

!!! warning "Важно"
    Если отключить автопродление, доступ к сервису будет остановлен после окончания текущей подписки.

---

## Частые ошибки

- Пользователь выбирает тариф без учёта количества магазинов.  
- Дополнительные опции включены случайно.  
- Промокод введён неверно или просрочен.  
- Не поставлена галочка условий оферты — оплата не проходит.
