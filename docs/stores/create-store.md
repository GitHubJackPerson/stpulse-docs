# Создание магазина в STPulse

[← Назад в раздел «Магазины и Ozon»](overview.md)

Эта инструкция помогает создать магазин в STPulse и подготовить его к загрузке данных из Ozon.

---

## 1. Откройте раздел «Магазины»

1. Нажмите на иконку профиля (правый верхний угол).
2. Выберите пункт **«Магазины»**.
3. Нажмите **«Добавить магазин»**.

---

## 2. Заполните данные магазина

В открывшейся форме укажите:

- **Название магазина** — любое.
- **Seller API Client ID**
- **Seller API Key**
- **Performance API Client ID**
- **Performance API Client Secret**
- **Система налогообложения**
- **Ставка УСН / НДС**
- **Дата начала начисления НДС**
- **Дата начала доп. страховых взносов**

!!! note "Где взять API-ключи?"
    Подробные инструкции см. в разделах:
    - [Seller API](seller-api.md)
    - [Performance API](performance-api.md)

---

## 3. Добавьте сервисные аккаунты Performance API

После ввода основного ключа создайте **5 сервисных аккаунтов** и добавьте их в STPulse.

!!! tip "Почему нужно 5 аккаунтов?"
    Несколько аккаунтов работают одновременно.  
    Это ускоряет загрузку рекламных отчётов в несколько раз.

Чтобы добавить аккаунты:

1. В STPulse нажмите кнопку **«+»** рядом с разделом Performance API.
2. Создайте дополнительные аккаунты в Ozon.
3. Вставьте ключи в интерфейс STPulse.

---

## 4. Сохраните настройки

Нажмите **«Сохранить»**.  
Магазин появится в списке доступных, а загрузка данных начнётся автоматически.

Статус загрузки:

**Профиль → Магазины → Ваш магазин → Импорт данных**

---

## Частые ошибки

- Неверная роль Seller API (нужна только **admin read only**).
- Перепутаны Seller API и Performance API.
- Ключ удалён в Ozon.
- Налоговые поля оставлены пустыми → отчёты будут неверными.
