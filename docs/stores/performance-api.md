# Performance API — подключение и ключи

Performance API используется для загрузки рекламных данных Ozon.  
Чтобы отчёты работали быстрее, рекомендуется создать несколько сервисных аккаунтов.

---

## 1. Где найти Performance API

1. Перейдите в **Кабинет Ozon → Профиль → Настройки → Performance API**.
2. Нажмите **«Создать аккаунт»**.
3. В списке аккаунтов выберите первый ID — например **90880088**.

---

## 2. Как создать ключ Performance API

1. Нажмите **«Добавить ключ»**.
2. Введите любое имя (например *STPulse Performance*).
3. Нажмите **«Добавить»**.
4. Скопируйте:
   - **Client ID**
   - **Client Secret**
5. Вставьте их в соответствующие поля в STPulse.

---

## 3. Почему нужно 5 ключей?

!!! tip "Главная причина — скорость"
    Несколько ключей работают параллельно.  
    Это ускоряет загрузку рекламных отчётов в 3–5 раз.

Чтобы добавить ключи:

1. В STPulse нажмите кнопку **«+»** в разделе Performance API.
2. Создайте в Ozon ещё несколько сервисных аккаунтов.
3. Вставьте ключи в STPulse.

---

## 4. Частые ошибки

- Добавлен только 1 ключ → отчёты загружаются медленно.
- Перепутаны Seller API и Performance API.
- Ключ удалён в Ozon.
- Аккаунт создан в другом магазине.
