# Seller API — подключение и настройка

Seller API — это интерфейс Ozon, через который STPulse получает данные о продажах, товарах и остатках.

Ниже описан полный процесс получения ключей Seller API.

---

## 1. Как найти Seller API Client ID

1. Перейдите в **Кабинет Ozon → Профиль → Настройки → Seller API**.
2. Найдите строку **Ozon ID**.
3. Скопируйте значение в поле **Seller API Client ID** в STPulse.

---

## 2. Как создать Seller API Key

1. Откройте **Профиль Ozon → Настройки → Seller API**.
2. Нажмите кнопку **«Создать ключ»**.
3. Укажите любое имя (например *STPulse Seller*).
4. Выберите только роль:

   ✔ **admin read only**

5. Нажмите **«Сгенерировать»**.
6. Скопируйте ключ и вставьте в поле **Seller API Key** в STPulse.

!!! warning "Важно"
    Показ ключа доступен только один раз.  
    Если вы закрыли страницу — создайте новый ключ.

---

## 3. Частые ошибки

- Выбрана роль **action read only**, а не **admin read only**.
- Ключ перепутан с Performance API ключом.
- Ключ был удалён в Ozon → данные перестают загружаться.
- Ключ скопирован не полностью.

---

## 4. Проверка подключения

После сохранения настроек магазин начнёт загружать данные автоматически.

Статус загрузки:

**Профиль → Магазины → Название магазина → Импорт данных**
