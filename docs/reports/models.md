# Модели

Модели — это система группировки товаров в отчётах STPulse.  
С помощью модели можно объединять несколько товаров в одну логическую группу, чтобы смотреть сводные показатели: продажи, остатки, себестоимость, прибыль и другие данные.

---

## 1. Для чего нужны модели

Модели помогают анализировать товар не только по отдельным артикулам, но и по объединённым группам. Это удобно, если:

- у товара есть много размеров одной модели;
- товары объединены в склейку внутри Ozon;
- у бренда много похожих товаров, и их нужно рассматривать вместе;
- один товар представлен в разных цветах или модификациях;
- нужно агрегировать аналитику по группе товаров, а не по каждому SKU отдельно.

!!! tip "Пример"
    Если у вас есть товар «Куртка зимняя» в размерах от XS до 5XL, вы можете задать им модель **«Куртка зимняя классика»** — и анализировать результаты по всей линейке сразу.

---

## 2. Показатели в таблице

Нажав кнопку **«Показатели»**, вы можете выбрать, какие столбцы отображать:

- **Категория**  
- **Название**  
- **Артикул**  
- **Модель**

Это помогает настроить таблицу под ваши задачи и убрать лишние поля.

---

## 3. Как назначить модели вручную

1. Откройте раздел **Отчёты → Модели**.
2. Выберите магазин.
3. Найдите нужный товар через поиск или фильтры.
4. В столбце **Модель** введите название модели вручную.
5. Нажмите Enter — значение сохранится.

Все товары с одинаковым названием модели будут объединены в одну группу во всех отчётах.

!!! warning "Важно"
    Модель должна совпадать **буква в букву** для всех товаров, которые вы хотите объединить.

---

## 4. Импорт моделей из файла XLSX

Если товаров много, удобнее обновлять модели через импорт.

1. Нажмите кнопку **«Импортировать»**.
2. Скачайте шаблон файла — кнопка **«Скачать шаблон с артикулами»**.
3. Откройте файл в Excel или Google Sheets.
4. Заполните два столбца:
    
    - **Артикул**  
    - **Модель**

5. Сохраните файл в формате XLSX.
6. Нажмите **«Импортировать»** и загрузите готовый файл.

После импорта модели сразу обновятся в таблице.

!!! tip "Совет"
    Если у всех товаров одной модели одинаковая часть названия, можно быстро заполнить столбец формулой или копированием.

---

## Пример

Предположим, вы хотите объединить товары:

- кардиган классический L  
- кардиган классический XL  
- кардиган классический XXL  

Вы создаёте модель:

**кардиган классика**


И присваиваете её всем трём артикулам вручную или через XLSX.  
Теперь в отчётах STPulse вы сможете смотреть:

- суммарные продажи по модели,
- общий остаток по всей линейке,
- себестоимость, прибыль, возвраты и другие показатели **по группе в целом**.

---

## Частые ошибки

- В названии модели есть лишний пробел → товары не объединяются.  
- Разные написания одной группы (например, «модель1» и «Модель 1») → создают две разные модели.  
- В XLSX указан неверный артикул → модель не применяется.  
- Заменили модель вручную, но забыли обновить файл импорта → при следующем импорте старые модели затрут новые.
