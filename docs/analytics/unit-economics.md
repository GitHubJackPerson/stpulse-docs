# Юнитка

Отчёт **«Юнитка»** показывает **юнит-экономику по каждому товару (артикулу)** за выбранный период.  
По сути, это та же «Операционка», но **без динамики по дням** и **только по товарам** — чтобы быстро понять, какие позиции зарабатывают, а какие работают в минус.

Отчёт помогает принимать решения по ассортименту, ценам и рекламе.

Данные сейчас доступны **только для Ozon**.

---

## Назначение и польза

Отчёт используют, чтобы:

- оценить прибыльность каждого артикула;
- найти убыточные товары;
- понять, какие позиции можно масштабировать;
- проверить, окупается ли реклама на уровне товара;
- сравнить товары внутри одной категории или модели.

---

## 1. Выбор магазина и периода

1. Откройте **Аналитика → Юнитка**.
2. Выберите магазин.
3. Укажите период показа данных.

Особенности периода:

- можно выбрать **любой период**;
- данные считаются **суммарно за период**;
- нет разбивки по дням — только итоговые значения.

Отчёт пересчитывается автоматически при изменении периода или магазина.

---

## 2. Структура отчёта

Отчёт строится **по артикулам** и может быть сгруппирован:

- по категориям;
- по моделям;
- по отдельным SKU.

Каждая строка — это **один товар (юнит)** с итоговыми показателями за период.

---

## 3. Основные показатели

В таблице доступны ключевые метрики юнит-экономики:

- **Заказы** — количество заказов товара.
- **Выручка** — сумма продаж.
- **Продажи** — количество выкупленных единиц.
- **Прибыль** — итоговая прибыль по товару.
- **Маржинальность** — доля прибыли в выручке.
- **ROI** — рентабельность вложений.
- **Себестоимость** — себестоимость товара.
- **Реклама** — рекламные расходы.
- **Расходы Ozon** — комиссии, логистика и другие списания.
- **Доставка до места выдачи** — расходы на доставку.

Цветовая подсветка помогает быстро находить:

- прибыльные товары (зелёный);
- убыточные позиции (красный);
- пограничные значения (жёлтый).

---

## 4. Фильтр «Показатели»

Кнопка **«Показатели»** позволяет:

- включать и отключать нужные метрики;
- создавать **собственные наборы показателей**;
- сохранять наборы для разных сценариев работы;
- настраивать отображение таблицы под задачу.

Также в фильтре можно:

- выбрать конкретные показатели для анализа;
- задать цвет выделения для любого показателя.

!!! tip "Подсказка"
    Настройте отдельный набор показателей для анализа прибыли и отдельный — для рекламы. Это ускоряет работу.

---

## 5. Категории и товары

Через фильтр **«Категории и товары»** можно:

- выбрать отдельные категории;
- анализировать конкретные модели;
- оставить в отчёте только нужные артикулы.

Это удобно, если:

- нужно проверить новую коллекцию;
- сравнить товары внутри одной линейки;
- проанализировать конкретный бренд.

---

## 6. Типовые сценарии использования

### Сценарий 1. Поиск убыточных товаров
- период: последний месяц;
- показатели: прибыль, маржинальность, расходы;
- цель: найти товары с отрицательной прибылью и принять решение  
  (повышать цену, отключать рекламу или выводить из ассортимента).

---

### Сценарий 2. Масштабирование прибыльных позиций
- отфильтровать товары с высокой маржинальностью и ROI;
- проверить объёмы продаж;
- использовать список для увеличения рекламного бюджета или закупки.

---

### Сценарий 3. Анализ рекламы по товарам
- включить показатели рекламы и ROI;
- сравнить товары с одинаковой ценой;
- понять, где реклама окупается, а где «съедает» маржу.

---

### Сценарий 4. Проверка новой линейки
- выбрать категорию или модель;
- сравнить прибыль и маржинальность;
- принять решение о дальнейшем развитии линейки.

---

## 7. Что важно помнить

!!! tip "Подсказка"
    Юнитка идеально подходит для **стратегических решений по ассортименту**.

!!! warning "Важно"
    В отчёте нет динамики по дням.  
    Если нужно понять, *когда* возникла проблема — используйте «Операционку».

---

## Частые ошибки

- Анализ только выручки без учёта прибыли.
- Игнорирование расходов на рекламу.
- Сравнение товаров из разных категорий без фильтрации.
- Использование слишком короткого периода.

---

Отчёт **«Юнитка»** отвечает на главный вопрос:  
**какие товары реально зарабатывают деньги, а какие — нет.**
