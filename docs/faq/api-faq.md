# Вопросы по API Ozon

---

## Почему STPulse не показывает данные?

Основные причины:

- Ключ Seller API удалён в Ozon.
- Performance API ключи созданы неверно.
- Ключи перепутаны между собой.
- Налоговые поля не заполнены.

---

## Сколько Performance API ключей нужно?

Мы рекомендуем **5 ключей**, чтобы ускорить загрузку отчётов.

---

## Можно ли использовать один Seller API для нескольких магазинов?

Нет.  
Для каждого магазина нужен отдельный Seller API.

---

## Что будет, если ошибиться в налогах?

Неверные налоги → неправильно считается прибыль.

---

## Потерял ключ. Где его посмотреть?

Нигде.  
Ключи в Ozon показываются один раз. Создайте новый.

---

## Почему реклама загружается медленно?

- Добавлен только один Performance API ключ.  
- Неверная роль Seller API.  
- Впервые загружаются большие отчёты.
